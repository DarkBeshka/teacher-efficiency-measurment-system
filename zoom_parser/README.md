# Парсер отчетов Zoom с опросами

Автоматизированный парсер для скачивания отчетов о конференциях Zoom, содержащих данные опросов.

## Возможности

- Автоматический вход в аккаунт Zoom
- Установка диапазона дат за прошедший месяц
- Фильтрация конференций по наличию опросов
- Автоматическое скачивание всех отчетов с учетом пагинации
- Сохранение файлов в локальную папку проекта

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Установите браузеры для Playwright:
```bash
playwright install chromium
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Отредактируйте `.env` и укажите свои учетные данные:
```
ZOOM_EMAIL=your_email@example.com
ZOOM_PASSWORD=your_password
```

## Использование

Запустите скрипт:
```bash
python zoom_parser.py
```

Скрипт автоматически:
1. Войдет в ваш аккаунт Zoom
2. Перейдет на страницу отчетов
3. Установит дату за прошедший месяц
4. Включит фильтр "содержит данные опроса"
5. Выполнит поиск
6. Скачает все отчеты в папку `zoom_reports/`

## Структура проекта

```
zoom_parser/
├── zoom_parser.py      # Основной скрипт парсера
├── requirements.txt    # Зависимости Python
├── .env                # Учетные данные (не коммитится)
├── .env.example        # Пример конфигурации
├── README.md           # Документация
└── zoom_reports/       # Папка с скачанными отчетами
```

## Примечания

- Браузер запускается в видимом режиме (headless=False) для отладки. Вы можете изменить это в коде.
- Скрипт ждет завершения каждой загрузки перед переходом к следующей.
- При возникновении ошибок скрипт попытается продолжить работу, где это возможно.

## Требования

- Python 3.7+
- Playwright
- Аккаунт Zoom с доступом к отчетам
